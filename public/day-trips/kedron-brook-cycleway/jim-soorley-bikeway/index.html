About Briscycle

Briscycle is a passion project by Ash Kyd. It originated back in 2009 when there was little or no information about (let alone cycling infrastructure) in Brisbane.

Today there are a bunch of resources, including Brisbane Council's cycling website, so this is aiming to be a bit more of a general resource.

The source code to this site is available on github which means with a little technical skill you can help update the site as well! But if you just have a question or comment, please feel free to get in touch by email.
